# 🎂 3D漫游游戏使用说明

## 📋 项目概述

这是一个为蟹老板生日派对创建的增强版网页应用，现在包含了一个全新的3D漫游游戏！

## 🎮 游戏特色

### 3D漫游游戏
- **完全3D环境**：基于Three.js构建的生日派对世界
- **第一人称控制**：WASD移动，鼠标控制视角，空格跳跃
- **收集要素**：金币和隐藏的生日礼物
- **精美场景**：包含蛋糕塔、气球、树木等装饰元素
- **特效系统**：收集时的粒子效果

### 原有游戏
- **接金币游戏**：经典的2D接金币游戏
- **蛋糕装饰**：创意蛋糕装饰游戏
- **许愿功能**：生日蜡烛许愿

## 🚀 快速开始

### 1. 本地运行
```bash
# 直接在浏览器中打开
open index.html
# 或
# 双击 index.html 文件
```

### 2. 部署到GitHub Pages
```bash
# 在项目目录中运行
./deploy.sh
```

## 🎮 3D漫游游戏操作指南

### 基本控制
- **W** - 向前移动
- **A** - 向左移动  
- **S** - 向后移动
- **D** - 向右移动
- **鼠标** - 控制视角
- **空格** - 跳跃

### 游戏目标
1. 收散落在场景中的金币（每个10分）
2. 找到5个生日礼物（每个50分）
3. 完成所有礼物收集即可获胜

### 场景元素
- 🎂 **蛋糕塔**：场景中的大型装饰建筑
- 🎈 **气球**：漂浮的彩色气球
- 🌳 **树木**：装饰用的树木
- 🎁 **礼物盒**：需要收集的生日礼物
- 💰 **金币**：散落在地面上的金币

## 🛠️ 技术特性

### 3D技术栈
- **Three.js**：3D图形库
- **PointerLockControls**：第一人称控制器
- **WebGL**：硬件加速3D渲染
- **阴影系统**：动态阴影效果
- **粒子系统**：收集特效

### 性能优化
- 使用LOD（Level of Detail）技术
- 优化的碰撞检测
- 高效的渲染循环

## 📱 兼容性

- **桌面浏览器**：Chrome, Firefox, Safari, Edge
- **移动设备**：支持触摸控制（部分功能）
- **性能要求**：建议现代浏览器，支持WebGL

## 🔧 自定义选项

### 修改游戏参数
```javascript
// roaming-game.js 中可修改的参数
const gameConfig = {
    coinCount: 20,        // 金币数量
    giftCount: 5,         // 礼物数量
    worldSize: 200,       // 世界大小
    playerSpeed: 0.5,     // 玩家移动速度
    jumpVelocity: 0.3     // 跳跃速度
};
```

### 自定义3D模型
- 可以替换 `createTree()`, `createCakeTower()` 等函数
- 添加新的3D模型和动画
- 自定义材质和纹理

## 🐛 故障排除

### 常见问题
1. **3D场景不显示**
   - 检查浏览器是否支持WebGL
   - 确保Three.js库正确加载

2. **控制不响应**
   - 点击"开始3D漫游"按钮激活控制
   - 确保鼠标在游戏区域内

3. **性能问题**
   - 降低粒子数量
   - 简化3D模型

### 调试方法
- 打开浏览器开发者工具
- 查看控制台错误信息
- 检查网络请求是否正常
