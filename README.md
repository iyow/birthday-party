# 🎂 蟹老板生日快乐！🎉

这是一个为蟹老板准备的生日庆祝网页，包含炫酷的动画效果和互动小游戏！

## ✨ 特性

- 🎬 炫酷的开场和落幕动画
- 🎮 接金币小游戏
- 🎂 生日蛋糕许愿功能
- 🌟 精美的生日祝福卡片
- 🎨 响应式设计，支持移动端

## 🎮 如何玩

### 接金币游戏
1. 点击"开始游戏"按钮
2. 移动鼠标控制钱袋子
3. 接住掉落的金币赚取分数
4. 不要让金币掉到地上，否则会损失生命
5. 生命值为0时游戏结束

### 许愿蛋糕
1. 点击"吹灭蜡烛许愿！"按钮
2. 看着蜡烛熄灭的动画
3. 愿望会发送到宇宙！✨

## 🚀 本地运行

1. 克隆或下载这个项目
2. 在浏览器中打开 `index.html` 文件
3. 享受生日派对！

## 📱 技术栈

- 纯HTML5、CSS3、JavaScript
- Canvas API用于游戏开发
- CSS动画和过渡效果
- Web Audio API（音效）

## 🎨 自定义

你可以根据需要修改以下内容：
- 生日日期：编辑 `index.html` 中的日期
- 游戏难度：修改 `game.js` 中的金币速度和生成频率
- 颜色主题：编辑 `style.css` 中的渐变色
- 生日祝福：修改 `index.html` 中的祝福卡片

## 🌿 分支管理

这个项目使用分支管理不同年份的生日页面：

- `main` - 主分支，包含最新的生日网页
- `2026` - 2026年农历二月二十五日生日页面
- `2027` - 2027年农历二月二十五日生日页面
- `2028` - 2028年农历二月二十五日生日页面

每个分支对应一年的生日，方便保存和回顾历年庆祝！
